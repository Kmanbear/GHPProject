The Caesar Cipher

The Caesar cipher is one of the oldest ciphers to have existed. It is also one of the simplest forms of encryption. It works by shifting each letter in a message by a certain number of letters. The key was simply the number of letters shifted right. In order to decrypt the encrypted message, one would simply need to shift the same number of letters to the left. Back then, this was a valuable encryption method because it would produce a ciphertext that would be unreadable. It would also take some testing to reveal the key, but frequency analysis led to the Caeser cipher easily being broken.
